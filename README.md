# Student Grade Analyzer

## Project Description
Student Grade Analyzer is a Python program that collects a student’s name and marks for multiple subjects, stores the data using lists and dictionaries, calculates the average score, assigns a letter grade, and determines pass or fail status. Functions are used to modularize tasks for clarity and reusability.

## Features
- Input student name and grades
- Store data in dictionaries and lists
- Calculate average marks
- Assign letter grades (A, B, C, D)
- Determine pass/fail based on average
- Display summarized results

## How to Run
1. Clone or download the project.
2. Open the Python file (`.py`) in any IDE or text editor.
3. Run the program.
4. Enter student details when prompted.
5. View the calculated average, grade, and result.

## Technologies Used
- Python 3
- Built-in functions: `input()`, `sum()`, `len()`
- Data structures: Lists and Dictionaries
- Functions for modularity
